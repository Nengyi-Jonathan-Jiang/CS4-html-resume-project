My favorite projects

  • Dysprosium

    My latest project. I am working on a full lexer + parser + optimizer + code generator + virtual machine for a new language I came up with, Dysprosium (I picked the name because it sounds cool, and it hasn't been used for much yet). Features a fully generic lexer (with a Regex -> NFA -> DFA compiler), a fully generic parser (built on the LR(1) parsing algorithm), a virtual machine (DysVM), etc. The language itself draws inspiration from C++, with memory safety mechanisms similar to Rust.

  • Reaction-Diffusion simulator

    A really cool simulation that I worked on. You can control the reaction parameters (reaction rate, diffusion radius, diffusion rates, feed rate, remove rate, etc.) and the speed of the simulation (steps per frame). For some combinations of parameters, complex structures reminiscent of cellular division can occur. I used compute shaders in WebGL to calculate the steps of the simulation.

  • Three body simulator

    Another really cool simulation that I worked on. Simulates three massive stars in 3d space interacting through gravity.

  • Display Math

    Math display thing I made, a bit similar to LaTeX. Not much more to be said about it. It's kinda janky and does weird stuff sometimes, but those are totally features, not bugs.

  • Go Capy Go!

    A retro-style platformer/puzzle game. Get the capybara to the candy! I made this with the help of Eva Li, Rounak Rai, and Santiago Espinoza for a game jam.

  • Five in a row (Gomoku, 五子棋)

    A classic board game. Get five stones in a row! The AI I made for this (classic techniques, no neural networks) is hard, but beatable. Good luck.

  • Klotski (华容道)

    A chinese sliding puzzle. Get the big piece labeled 曹 to the bottom. These puzzles are harder than they looks -- many require 50+ moves, with the last one requiring more than 130!

  • Rush Hour

    A Rush Hour game made with the Unity Game Engine. Pretty self-explanatory.

  • Maze Game

    A maze game featuring keys and obstacles. All levels are randomly generated, so some may be easier than others. However, all are guaranteed solvable.

  • Circle good, Square bad

    A bullet hell game. Eat the circles and avoid everything else! Best played on a touchscreen device, although mouse is okay. Touchpad highly discouraged.
